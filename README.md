# Student Grade Tracker

## Description
Java project to input and manage student grades. Calculates average, highest, and lowest scores.

## Features
- Input student names and scores
- Calculate average, highest, lowest
- Display summary report

## How to Run
1. Open project in any Java IDE (IntelliJ/Eclipse)
2. Run StudentGradeTracker.java
3. Follow console instructions

## Technologies
- Java
- ArrayList
- File I/O (optional)
